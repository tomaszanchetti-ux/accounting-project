# Executive Demo Script V2 (2 Minutes Max)

## Objective

Short walkthrough for the simplified **Accounting Reconciliation MVP**.

Audience:

- finance leaders
- controllers
- accounting managers
- operations or transformation sponsors

Target length:

- `2 minutes maximum`

Core message:

> This MVP turns payroll reconciliation into a simple guided workflow: load payroll, enter expected totals, run the check, review the summary, and export evidence.

---

## Demo Flow

### 1. Opening — 15 seconds

**What to show**

- home / setup screen

**What to say**

> This is a simplified reconciliation MVP for accounting teams.  
> Instead of working across multiple spreadsheets and manual checks, the user follows one guided flow from input to explanation to exportable evidence.

---

### 2. Inputs and Run — 30 seconds

**What to show**

- `Payroll file`
- `Expected Totals`
- run button

**What to say**

> The workflow starts with just two critical inputs.  
> First, we upload the payroll file.  
> Second, we enter the expected totals by concept, which is often how accounting receives the control numbers from HR or another source.  
> Then we launch the reconciliation run in one click.

---

### 3. Results Summary — 30 seconds

**What to show**

- summary KPIs
- concepts table
- highlight `MEAL_VOUCHER`

**What to say**

> Once the run is complete, the summary shows the overall picture immediately.  
> We can see which concepts reconciled, which ones did not, and where the main differences are.  
> In this case, `MEAL_VOUCHER` stands out, so we open it directly for investigation.

---

### 4. Deep Dive — 30 seconds

**What to show**

- expected vs observed
- summary statement
- top causes
- evidence table

**What to say**

> The Deep Dive is where the MVP becomes really useful.  
> It does not only flag a variance. It explains the likely drivers behind it.  
> Here we can see the expected amount, the observed amount, the summary explanation, and the top causes, such as out-of-period records, duplicate records, or unmapped concepts.  
> Right below that, the evidence table lets the reviewer validate those findings at row level.

---

### 5. Export — 15 seconds

**What to show**

- `Export evidence CSV`
- `Export PDF report`

**What to say**

> Once the reviewer understands the issue, the result can be exported immediately.  
> The team can download the detailed evidence as CSV or generate a PDF report that is easier to share with management, accounting, or audit stakeholders.

---

## Closing Line — 10 seconds

**What to say**

> So the value of this MVP is simple: fewer steps, faster reconciliation, clearer explanations, and a direct path from variance to evidence in a workflow that feels intuitive for finance users.

---

## Presenter Notes

### Keep the focus on:

- simplicity
- intuitive flow
- speed to insight
- explainability
- exportable outputs

### Avoid spending time on:

- backend logic
- technical architecture
- deployment
- low-level data model details

### Recommended narrative:

1. show the two inputs
2. run the reconciliation
3. land on the summary
4. open `MEAL_VOUCHER`
5. show evidence and exports
6. close on business value

---

## Ultra-Short Version

> This MVP gives accounting teams a simple way to reconcile payroll against expected totals.  
> The user uploads payroll, enters expected totals, runs the reconciliation, reviews a clear summary, opens a concept-level Deep Dive, and exports the supporting evidence.  
> The result is a much faster and more intuitive path from mismatch to explanation.
